
From src folder, please follow these steps:

1. Install project:

npm install

2. Create datasources.json

With PostgreSQL: 

cp server/datasources.json.template  server/datasources.json

With MySQL:

cp server/datasource.json.mysql.example  server/datasources.json

3. Customize datasources.json with your database

4. Prepare DB: 
With PostgreSQL:

createdb JRSP

With MySQL:

// CREATE DATABSE JRSP

5. Setup Database

node server/bin/automigrate
